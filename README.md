# School_gestion-EGest

EGest – Plateforme Intelligente de Gestion d’Établissements Scolaires

📖 Présentation

EGest est une plateforme moderne de gestion d’établissements scolaires conçue pour simplifier l’administration, améliorer la communication et automatiser les tâches académiques grâce aux technologies web, mobile et à l’intelligence artificielle.

Le projet vise à offrir une solution complète pour les écoles, collèges, lycées, universités et centres de formation.

⸻

🎯 Objectifs

* Digitaliser la gestion scolaire.
* Centraliser toutes les données de l’établissement.
* Réduire les tâches administratives.
* Faciliter la communication entre les acteurs.
* Exploiter l’Intelligence Artificielle pour assister les utilisateurs.
* Fournir une plateforme rapide, moderne et sécurisée.

🚀 Installation

1. Cloner le projet

git clone https://github.com/votre-utilisateur/EGest.git
cd EGest

⸻

2. Backend

cd backend
python -m venv venv
source venv/bin/activate
# Windows
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload

⸻

3. Frontend

cd frontend
npm install
npm run dev

⸻

4. Mobile

cd mobile
npm install
npx react-native run-android

⸻
